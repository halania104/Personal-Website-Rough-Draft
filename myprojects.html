Weather App

My weather app consists of basic HTML5, CSS, and JavaScript. Users can search different cities and get the current weather information, as well as the weather forecast for the next sevearl days. The app has icons that match to the forecast for each day, with a background of bright colors. This project required me to apply the use of the following: Search Engines, API integration, wind speed, a weather description, and the use of corresponding weather icons.
